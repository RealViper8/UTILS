Help

Here is the help page for Utilities Terminal CLI

Start Arguments

You can use this arguments by opening a terminal in the current project directory and typing python main.py <-argument->

  • --help : Shows the rest of arguments that you can use
  • --database : Switches directly to the database mode
  • --gcw <-WlanSSID-> : Shows you the passwort of the saved wifi
  • --show-profiles : Shows you all the saved wifi profiles
  • --engine : Start the 3d engine that lets you view some models in assets/roboter.obj etc
  • --wsl : Lets you start the wsl aka Distro if you have it installed

More info at: githubreadme

Commands

Our terminal commands are designed and easy to understand for a diverse range of users

  • help This command shows you the list of available commands
  • dw <-URL-> Download File using url and proxy if needed
  • pt <-TEXT-> Change the 'Terminal Text'
  • iwshow Shows the information that websites would see or could get from you
  • proxy Use and change proxy when browsing with utils
  • host-l <-PORT-> Host a local TCP Server with a custom port
  • host-w Host a wireless local area server
  • encoder Launch golang base64 encoder/decoder
  • exit Exits with code 0
  • database Change to database mode
